Chapter 2 Introduction

Again, some placeholders for some handle thing we could add-in later.

You can label chapter and section titles using {#label} after them, e.g., we can reference Chapter 2. If you do not manually label them, there will be automatic labels anyway, e.g., Chapter ??.

You can write citations, too. For example, we are using the bookdown package (Chuck 2019) in this sample book, which was built on top of R Markdown and knitr (Xie 2015).
